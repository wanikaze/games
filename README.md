# 🕹️ Wanikaze's Game Arcade

Welcome to my personal collection of handcrafted, browser-based electronic pocket games! 

This project was built from the ground up using HTML, CSS, and vanilla JavaScript—all without me knowing how to code, completely directed through AI collaboration.

## 🎮 The Games

1. **Whac-A-Mole:** A pure test of reflexes. Bonk the lit moles and avoid the dark ones.
2. **Gator Panic:** A frantic landscape whack-a-gator game with adaptive difficulty.
3. **Pop It! Pro:** Memorize light patterns across 20 unlockable levels.
4. **Pop It! Color Mix:** Test your color recognition and timing under pressure.
5. **Pop It! Go:** Fast-paced, bite-sized bubble popping minigames.

## 🚀 Features
* **No Dependencies:** 100% vanilla HTML, CSS, and JS.
* **Responsive:** Designed heavily for mobile touch screens, but fully playable with a mouse.
* **Custom Audio:** Sound effects are generated entirely using the Web Audio API (no external sound files to download!).
* **PWA Ready:** Add the arcade to your phone's home screen to play in full-screen standalone mode.

## 💡 How to Play
Just open `index.html` in any modern web browser, or visit the live GitHub Pages link!
